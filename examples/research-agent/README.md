# 리서치 에이전트 템플릿 (가설 생성)

팀별 리서치 AI Agent를 만들 때 출발점으로 쓰는 템플릿입니다.
이 폴더를 통째로 복사해 팀 작업 폴더로 사용하세요.

> **스코프 (Day 4 3세션 완주)**
> - **목표** — corpus의 **gap에서 'rationale 갖춘 새 연구 가설'을 발견·제안** (질문에 답하는 게 아님)
> - **corpus 기반** — `sources/` 같은 **문서 폴더**에서 자료 파악 (오픈 웹 크롤링·벡터DB 불필요)
> - **파이프라인** — 자료 파악 → **gap·모순 발견** → **가설 생성(다수)** → **평가·우선순위** → 제안
> - **산출물** — **rationale 갖춘 새 가설 제안서** (가설 + 근거·새로움·중요성 + 검증 아이디어 + 우선순위)
> - 작게 시작(MVP: 자료 파악 → gap 1개 → 가설 1개) → 다수 생성·평가로 확장

## 폴더 구조
```
research-agent/
  CLAUDE.md                 # 에이전트 작동 규칙 (자료파악→gap→가설생성→평가·제안)
  research-plan-template.md # 팀 기획서 (분야·발견 목표·범위)
  run-log-template.md       # 실행 기록 (무엇을 시도/관찰했나)
  collect/                  # ★ 논문 수집(PubMed) 스크립트 + 가이드 (Day 4 오전)
  sources/                  # 자료 폴더 — 교육용 가상 자료 6편 (실수업은 papers/로 교체; 네트워크 없는 리허설용으로도 사용)
  worksheets/               # 단계별 작업 시트 — 논문 정리표 · gap 목록 · 가설 평가표
  output/                   # 결과물 — 가설 제안서·분석 노트 양식 + 완성 예시(SAMPLE-...)
```

## 워크플로우 (Day 3~5)
1. **기획** (Day 3): `research-plan-template.md` 를 채워 분야·발견 목표 확정
2. **구현** (Day 4 오전): `CLAUDE.md` 규칙대로 자료파악→gap→가설생성→평가 흐름 구성 (MVP부터)
3. **실행/분석** (Day 4): `sources/` 자료로 돌려 가설 생성, `output/` 에 평가·분석 정리
4. **제안/발표** (Day 5): `output/report-template.md` 로 가설 제안서·발표자료 작성

## 필요 조건 (이 폴더만 있으면 되나?)

**이 폴더는 자립형(standalone)입니다.** 압축을 풀고 아래만 갖춰지면 바로 실행됩니다.

| 필요한 것 | 언제 쓰나 | 없으면 |
|-----------|-----------|--------|
| **Claude Code 설치 + 로그인** | 전 과정 (Day 2에서 설치함) | 필수 — 이게 있어야 실행 |
| **Python 3** | `collect/collect_pubmed.py` 실행 시에만 | 논문 수집 스크립트만 안 됨 (나머지는 됨) |
| **인터넷 연결** | PubMed로 실제 논문 수집할 때만 | `sources/` 가상 자료로 오프라인 연습 가능 |

> - 별도 설치(pip 등)는 **없습니다** — 수집 스크립트는 표준 라이브러리만 씁니다.
> - `CLAUDE.md`(작동 규칙)는 이 폴더에서 `claude` 를 켜면 **자동으로 로드**됩니다.
> - `sources/` 덕분에 인터넷 없이도 자료파악→gap→가설 전 과정을 연습할 수 있습니다.

## 시작하기
```bash
# 압축을 푼 폴더 안으로 이동
cd research-agent
claude
```
그다음(예): "research-plan.md 를 읽고, sources/ 자료에서 **모순·공백(gap)을 찾아 새 가설을 여러 개** 제안해줘. 각 가설에 근거(출처)와 왜 가치 있는지(rationale)를 달고, 우선순위를 매겨줘. 이미 잘 알려진 건 빼고."

> ⚠ `sources/` 는 **교육용 가상 자료**입니다. 실제 프로젝트에선 진짜 자료/웹검색으로 바꾸세요.
