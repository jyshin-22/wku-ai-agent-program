# gap 목록 (Gap List) — 작업 시트

> Day 4 오후1 'gap 발견' 단계 산출물. 정리한 논문들(논문 정리표)에서
> **빈틈(gap)**을 찾아 정리한다. gap은 새 가설의 원천이며, 각 gap에는
> **어느 논문에서 비롯되었는지 근거를 반드시 표기**한다.

## gap 유형 (Day 3 단계④ 생성 방식과 대응)
- **모순/충돌** — 두 논문의 주장이 다른 지점
- **미검증** — 주장은 있으나 근거·재현이 약한 지점
- **빠진 관점** — 특정 집단·조건·조합이 다뤄지지 않은 지점
