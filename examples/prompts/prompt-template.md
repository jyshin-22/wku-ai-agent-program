# 프롬프트 템플릿 (일반)

> 좋은 프롬프트 = **역할 + 맥락 + 작업 + 형식 + 제약**

## 기본 골격 (복사해서 채우기)
```
[역할]  너는 ___ 전문가야.
[맥락]  상황: ___ / 대상 독자: ___ / 목적: ___
[작업]  ___ 을(를) 해줘.
[형식]  출력 형식: ___  (예: 표, 불릿 5개, 200자 요약, 코드 블록)
[제약]  ___ 는 하지 마 · 출처를 함께 · 모르면 "모른다"고
```

## 예시 1 — 요약
```
너는 의학 논문을 쉽게 풀어주는 전문가야.
아래 텍스트를 비전공자도 이해할 수 있게 200자 이내로 요약해줘.
핵심 3가지를 불릿으로, 어려운 용어는 괄호로 풀어서.
근거가 없는 내용은 추가하지 마.
---
[여기에 텍스트 붙여넣기]
```

## 예시 2 — 코드(바이브코딩)
```
Python으로 CSV 파일을 읽어 항목별 합계를 출력하는 스크립트를 만들어줘.
- 입력: expenses.csv (열: 날짜, 항목, 금액)
- 출력: 항목별 합계 표 + 총합
- 실행 방법도 알려줘. 외부 라이브러리는 표준 라이브러리만.
만든 뒤 직접 실행해서 동작을 확인해줘.
```

## 체크리스트
- [ ] 목표가 한 문장으로 분명한가?
- [ ] 맥락(누구·무엇·왜)을 줬는가?
- [ ] 원하는 출력 형식을 명시했는가?
- [ ] 검증 가능한가? (출처/근거/실행 확인 요구)
- [ ] 한 번에 너무 많은 걸 시키지 않았는가? (작게 쪼개기)
